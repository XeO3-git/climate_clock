# climate_clock
A clock showing various climate data points, using data from https://climateclock.world/ and https://climatetrace.org/
The intent is to create a live updating clock that displays the following:
- How long until we hit 1.5 degrees warmed above pre-industrial temperatures.
- The percent of the world's energy created by renewables.
- How many gigatons of carbon dioxide have been released into the atmosphere.
- The current number of degrees Celsius warmed since pre-industrial temperatures.

I plan on using a Raspberry Pi connected to a repurposed iPad screen to display this data.
